Caja de herramientas para las humanidades digitales

En esta caja de herramientas incluimos solamente algunas de las aplicaciones más populares y de uso frecuente. Si deseas que incluyamos una herramienta en particular, solamente debes sugerirla como un issue en el repositorio de Github del curso.

Zotero

Gestor bibliográfico de código abierto creado por el RR-CHNM y desarrollado por la Corporation for Digital Scholarship.

Omeka

Software para la creación de colecciones y exhibiciones web diseñado para las bibliotecas, archivos, museos y otras organizaciones similares.

Tropy

Software de escritorio para organizar, describir, anotar y compartir imágenes y fotografías de archivos.

Recogito

Servicio web para realizar anotación semántica de textos e imágenes, tanto en proyectos individuales como colaborativos.

Transkribus

Servicio web para la transcripción automatizada (HWR) de documentos manuscritos.

Palladio

Servicio web para la visualización de datos tabulares en forma de red, mapa o galería.

TimelineJS

Servicio y librería de JavaScript para la creación y publicación de líneas de tiempo.

Mallet

Aplicación de escritorio para procesamiento de lenguaje natural con algoritmos de inteligencia artificial.

AntConc

Aplicativo de escritorio para el análisis de concordancias y textos.

OpenRefine

Aplicación de escritorio para limpieza de datos o data wrangling.

displaCy Named Entity Visualizer

displaCy Named Entity Visualizer es una herramienta que permite reconocimiento de entidades nombradas (personas, organizaciones, localidades), capaz de analizar varios idiomas.
